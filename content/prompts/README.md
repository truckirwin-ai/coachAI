# Prompt Registry

Versioned prompt templates. See SPEC.md §12.3

## Prompt Types
- system/ — Base coaching persona, behavioral rules, safety constraints
- skill-scaffolds/ — Per-skill coaching frameworks
- evaluation/ — Roleplay character definitions, scoring rubrics
- refreshers/ — Weekly reinforcement patterns
- intent-detection/ — Session intent classification
- summaries/ — Session summary generation
- pattern-detection/ — Behavioral pattern analysis
